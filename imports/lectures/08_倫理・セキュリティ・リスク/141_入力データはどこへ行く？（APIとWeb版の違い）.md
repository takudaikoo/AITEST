# 141. 入力データはどこへ行く？（APIとWeb版の違い）

## 1. はじめに
「ChatGPTやGeminiに入力した会社のデータ、勝手に学習されて漏れたりしない？」
この不安を解消するには、**「Web版（ブラウザ）」と「API（法人向け）」**のデータあつかいの違いを理解する必要があります。
結論から言うと、**無料のWeb版は学習されるリスクがあり、API/法人版は学習されません**。

![APIとWeb版のデータフロー](./../attachments/08_141_data_flow_1769361018908.png)

## 2. Web版（コンシューマー向け）
私たちが普段ブラウザで使う無料のGeminiやChatGPTは、サービス向上のために**入力データがAIの学習（再トレーニング）に使われる**規約になっていることが多いです。
つまり、ここに入れた機密情報は、将来AIが他の誰かに喋ってしまう可能性があります。

## 3. API・法人版（エンタープライズ）
Gemini Advanced（Enterpirse版）やAPI経由の利用は、**「入力データを学習に使わない（ゼロデータリテンション）」**という契約が結ばれます。
企業で使うなら、必ずこちらのアカウントや環境を使うのが鉄則です。

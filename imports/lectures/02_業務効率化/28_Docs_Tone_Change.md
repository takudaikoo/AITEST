# 28. Googleドキュメント × Gemini：文章のトーン変更（丁寧・断定的・親しみ）

## 1. はじめに
「内容は合っているけど、言い方がちょっとキツイかな…」
「もっと格式高い表現にしたいな」
そう思った時、全文を書き直す必要はありません。Geminiに**トーン（口調）**を変えてもらいましょう。

![文章のトーン変更機能](./../attachments/02_28_docs_tone.png)

## 2. 操作方法
1.  トーンを変えたい文章を選択します。
2.  Geminiアイコン（または右クリックメニュー）から「**Tone（トーン）**」を選択します。
3.  以下のような選択肢が表示されます。
    *   **Formal（フォーマル）**: より礼儀正しく、ビジネスライクに。
    *   **Casual（カジュアル）**: 親しみやすく、柔らかく。
    *   **Concise（簡潔に）**: 無駄を削ぎ落として短く。

## 3. 活用シーン
*   **社内向け**資料を**社外向け**に書き換える時（Casual → Formal）。
*   **専門的な説明**を**一般向け**にする時（Formal → Casual）。
*   **冗長な報告書**を**要約**する時（Concise）。

---

## 4. まとめ
「何を言うか（内容）」と「どう言うか（トーン）」は別の作業です。
内容は自分で考え、トーンの調整はAIに任せる。これでTPOに合わせた文書が一瞬で完成します。

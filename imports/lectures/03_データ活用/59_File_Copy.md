# 59. 毎日決まった時間にドキュメントをコピーする自動化

## 1. はじめに
「毎日、日報のテンプレートをコピーして、今日の日付に変えて…」
「毎週、定例会議の議事録ファイルを準備して…」
この数分の作業も、年間で言えば何時間もの損失です。
GASを使えば、**ファイルの複製・リネーム・フォルダ移動**まで全自動化できます。

![ファイル操作の自動化](./../attachments/03_59_catchphrase_generation_1769361585763.png)

## 2. 自動化プロンプト
> **プロンプト例**:
> 「Googleドライブにある『議事録テンプレート（ファイルID: xxx）』をコピーして、
> 指定のフォルダ（フォルダID: yyy）に保存したいです。
>
> ファイル名は『**2025-01-01_定例会議**』のように、**実行した日の日付**を自動で入れてください。
> これを行うGASコードを書いて」

## 3. ファイルIDの取得方法
コードを書くには、ファイルやフォルダの「ID」が必要です。
これもGeminiに「GoogleドライブのファイルIDってどこを見ればわかるの？」と聞けば、「URLの `d/` の後ろの部分ですよ」と教えてくれます。

---

## 4. まとめ
「ルーチンワーク（単純作業）」は、本来人間がやるべき仕事ではありません。
クリエイティブな仕事に時間を使うために、定型作業はAIとスクリプトに任せましょう。

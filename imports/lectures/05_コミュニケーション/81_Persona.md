# 81. 顧客ペルソナ（架空の顧客像）の詳細設定と対話シミュレーション

## 1. はじめに
「お客様の気持ちになって考えろ」と言われても、自分以外の人間になるのは難しいものです。
Geminiに**「架空の顧客（ペルソナ）」**になりきってもらい、対話をすることで、顧客視点をリアルに体験できます。

![ペルソナとの対話](./../attachments/05_81_python_intro_concept_1769361989371.png)

## 2. ペルソナ設定プロンプト
> **プロンプト例**:
> 「あなたは以下の人物になりきってください。
> *   名前：佐藤さん（35歳、IT企業勤務、既婚、子供1人）
> *   性格：慎重派、コストパフォーマンス重視
> *   悩み：最近仕事が忙しく、家事の時間が取れない
>
> この佐藤さんとして、私の提案する『家事代行サービス』について、**懸念点や質問**を投げかけてください」

## 3. ロールプレイング
AI佐藤さん：「便利そうですけど、他人を家に入れるのはちょっと不安ですね。セキュリティはどうなっていますか？」
あなた：「身元保証されたスタッフのみ派遣します」
AI佐藤さん：「それでも、物が壊されたりした時の補償は？」

このように、実際に話すことで「あ、そこが気になるんだ」という気付きが得られます。

---

## 4. まとめ
想像上の「お客様」ではなく、対話可能な「AIお客様」を持つこと。
これにより、あなたの提案やサービスは、独りよがりではない「顧客に刺さる」ものへと進化します。

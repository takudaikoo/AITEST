# 173. 開発：ダミーデータ（JSON）の大量生成

## 1. はじめに
アプリの開発中、「テスト用のデータ」を作るのに時間を費やしていませんか？
「ユーザー1、ユーザー2…」みたいな適当なデータでは、本番想定の検証ができません。
Geminiなら、**「本物っぽいダミーデータ」**を一瞬で大量生成できます。

![ダミーデータ生成](./../attachments/09_173_dev_dummy_json_1769364917146.png)

## 2. JSON生成プロンプト
> **プロンプト例**:
> 「ECサイトのユーザー情報のダミーデータを生成して。
> 形式はJSONで、以下のフィールドを持つこと。
> 1.  id (UUID)
> 2.  name (日本人氏名)
> 3.  email (ランダム)
> 4.  purchase_history (過去の購入品リスト配列)
> **データ数は10件。なるべくリアルな内容**で」

## 3. 異常データの生成
「メールアドレスの形式が崩れているデータ」「名前が空欄のデータ」など、**テストでエラーを起こすための意地悪なデータ**も、AIなら簡単に作れます。

## 4. まとめ
データ作りは単純作業ですが、品質に関わる重要な工程です。
AIに任せれば、あなたはテストロジックの構築そのものに集中できます。
